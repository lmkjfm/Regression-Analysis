# Statistical Analysis in R: Regression Models

This repository contains R scripts solving three major statistical problems:
1. **Student Score Prediction:** Analyzing the correlation between preliminary and final exam grades.
2. **Consumer Tip Analysis:** Modeling the relationship between bill totals and tip amounts.
3. **Agricultural Yield Study:** Assessing the impact of Ozone levels on crop production using ANOVA.

### Features
- Simple Linear Regression
- Pearson Correlation Coefficients
- Hypothesis Testing (p-values)
- 95% Confidence Intervals
- ANOVA for models through the origin
